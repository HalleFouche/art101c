Lab 4 - Everyday Task + Game Pseudocode

Challenge

In this lab our assignment was to pseudocode an unfamiliar everyday task and a simple computer game. Then we converted the pseudocode into comments.

Problems

The first problem we ran into was understanding the task we were presented with, but we also had trouble understanding how to break down the tasks logically at first. Breaking everything down into high-level tasks did help us organize a lot.

Reflection

At first I was worried this lab would be challenging but It turned out to be pretty straight forward and easy! I personally was having a rough week with my personal life, so I did not put much extra effort into the assignment.

Results

Everyday Task: Changing a Tire


			1. Park the car safely
				a. Find a flat surface
				b. Turn on hazard lights
				c. Apply parking brake
			2. Gather necessary tools
				a. Spare tire
				b. jack
				c. Lug wrench
			3. Loosen lug nuts slightly
			4. Jack up the car
			5. Remove lug nuts and take off the flat tire
			6. Mount the spare tire
			7. Tighten the lug nuts by hand
			8. Lower the car using the jack
			9. Fully tighten the lug nuts with the wrench
			10. Put away tools and flat tire
					

					
Game Task: Rock-Paper-Scissors

					

			1. Display game options to player (rock, paper, scissors)
			2. Get player input 
			3. Randomly select computer choice
			4. Compare player and computer choices
				a. If same, it's a tie
				b. If player wins, display win message
				c. If computer wins, display lose message
			5. Ask if player wants to play again
